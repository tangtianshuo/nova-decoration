# 装修企业展示与二维码分享平台 - 文档总览

## 1. 项目目标

本项目用于服务家装/工装装修公司，实现以下核心能力：

- 公司用户上传图片、视频到 Cloudflare R2。
- 支持录入 Bilibili 视频外链并纳入同一展示页。
- 系统为展示页生成二维码，扫码后可查看公司介绍、产品介绍、视频/图片内容、联系方式。
- 二维码长期有效，并支持失效后跳转兜底页，避免用户扫码出现空白或 404。

## 2. 推荐技术栈

- 前端：Vercel 托管的 Next.js（偏静态 + 少量动态请求）
- 轻后端：Cloudflare Workers（无传统服务器）
- 文件存储：Cloudflare R2
- 数据层：Cloudflare D1（主） + KV（短链缓存，可选）
- 边缘安全：Cloudflare DNS、WAF、Rate Limit、Turnstile

## 4. 建设里程碑（建议）

- 第 1 周：需求冻结、UI 原型、数据库建模
- 第 2 周：账号体系、公司信息、素材上传（R2 直传）
- 第 3 周：展示页编辑/发布、二维码生成、Bilibili 外链
- 第 4 周：统计埋点、兜底页、安全加固、联调与上线

## 5. 开发顺序建议（最小可用）

1. 完成数据库与 Worker 基础框架（鉴权 + 公司/页面 CRUD）。
2. 打通 R2 上传链路（签名 URL -> 前端直传 -> 回写素材表）。
3. 完成公开展示页读取与短链跳转（`/q/:code`）。
4. 接入二维码生成下载与失效兜底页跳转逻辑。
5. 最后完善统计、风控、监控与运维流程。

## 6. 非目标（首期不做）

- 复杂视频转码（可后续由 Cloudflare Stream 或第三方接入）
- 多级审批工作流
- 多语言国际化
- 高级 BI 报表

## 7. 成功标准（首期）

- 企业可在后台创建并发布展示页。
- 展示页可包含：公司介绍、产品介绍、联系方式、图片、视频、Bilibili 外链。
- 二维码扫码成功率 > 99.9%（在可控网络条件下）。
- 二维码可长期访问，失效时可跳转兜底页。
